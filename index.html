Projects

Things I’ve built

Hallway Duty — Janitor Simulator

A small 3D simulator where you play as a high-school janitor. I modeled assets in Maya, then textured and built gameplay in Unity. The scene uses reflective lighting for shiny surfaces, with interactive tasks and simple level scripting.

  • 3D assets modeled in Maya; exported and optimized for Unity.
  • Reflective materials and lighting passes for floors/fixtures.
  • Lightweight interaction loop (pickup, clean, dispose) with triggers.

UTM CPR Simulator — VR Training

A VR immersive environment modeled after a UofT lecture hall. The player practices AED placement and CPR compressions on interactive dummies. Built for the Oculus Quest 2 with controller interactions and clear in-headset prompts.

  • Real-world scale layout and collision for the lecture hall.
  • Grab & place interactions for AED pads; timing cues for compressions.
  • Runs on Quest 2; tuned for comfortable frame rates.

Office Outbreak — Zombie Wave FPS

Fast-paced wave-based FPS set in an office. Fight off escalating waves of zombies using multiple weapons, ammo pickups, and basic AI pathfinding. Built in Unity with C# for player controls, enemy behavior, and the wave system.

  • Weapon switching, reloads, and damage system.
  • Wave spawner with scaling difficulty and pacing.
  • Zombie AI navigation with chase/attack states.

Discord School Schedule Assistant (LLM Bot)

A Discord-based scheduling assistant that helps students build personalized weekly study plans. The bot asks structured questions about course load, availability, and priorities, then uses the 4o-mini API to generate a realistic time-management schedule tailored to the user.

  • Built with Python using the Discord API for real-time interaction.
  • Integrates OpenAI's GPT-4o mini to reason over constraints and priorities.
  • Generates adaptive weekly schedules based on user input and availability.
  • Designed for scalability and future features (calendar export, reminders).
